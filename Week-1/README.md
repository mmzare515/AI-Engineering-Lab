# Week 1 — Foundations of AI Engineering

This week focuses on building the foundational development skills and workflows used in modern AI engineering projects.

The goal is to establish a clean Python development environment, learn proper project organization, and begin working with tools commonly used in LLM application development.

---

## Topics Covered

### Day 1 — Development Environment & GitHub Setup

- Python virtual environments (`venv`)
- Git & GitHub basics
- VS Code / Cursor setup
- Project structure organization
- `.gitignore` basics
- Installing Python packages

### Day 2 — `.env` & Secrets Management

- Environment variables
- Using `.env` files
- Loading secrets securely with `python-dotenv`
- Accessing variables using `os.getenv()`
- Creating `.env.example`
- API key security best practices

### Upcoming Topics

- Pydantic basics
- Hugging Face & LLM API calls
- Streamlit UI basics
- LangChain introduction
- Vector databases
- RAG fundamentals
